# 📚 The Book Haven

The Book Haven is a full-stack online bookstore developed as a **Web Technologies** project. The application provides a complete online shopping experience where users can browse books, register and log in, manage their shopping cart, place orders, and subscribe to newsletters. An admin dashboard is included to manage the system efficiently.

## ✨ Features

* User Registration & Login
* Browse Book Collection
* Search Books
* Shopping Cart
* Checkout & Order Placement
* Newsletter Subscription
* Admin Dashboard
* Secure Database Integration

## 🛠 Technologies

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* phpMyAdmin
* Node.js (Database Connection)

## 🚀 How to Run

1. Clone the repository.
2. Import `bookshop.sql` into MySQL using phpMyAdmin.
3. Configure the database credentials in `database.php`.
4. Place the project inside your XAMPP/WAMP `htdocs` directory.
5. Start Apache and MySQL.
6. Open the project in your browser.
